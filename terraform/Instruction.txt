Terraform Instruction:
======================

- MUST! - aws ecr create-repository --repository-name vprofile/app (Mutable Image)
- IAM User: vprofile-jenkins-ecr 
    - AmazonEC2ContainerRegistryFullAccess
    - AmazonECS_FullAccess
    
- Run: 
        terraform init
        terraform apply

- After Apply Terraform will Create:
    - VPC
    - Subnets
    - RDS
    - ECS Cluster: Fargate and using placeholder image: amazon/amazon-ecs-sample
      (After Running Jenkins pipeline: the placeholder will be replaced with our custom vprofile app image from ECR)
    - ALB
    - ECS Service

- Check ECS Service state:
    - aws ecs describe-services --cluster vprofile-cluster --services vproapp-service
      (Should show formated info about the ECS Service in JSON format)
    
- Copy Paste private_subnets variable value to Jenkins pipeline.Jenkinsfile
- Copy Paste ecs_security_group variable value to Jenkins pipeline.Jenkinsfile




Connect to EC2 instance using SSM:
- aws ssm start-session --target i-09f5132500ba58705 (Instance ID of Bastion Host)
