# SonarQube Code Quality

<br>

## 📌 Overview

<br>

SonarQube is used for static code analysis in the CI/CD pipeline.

<br>

## Analysis Flow

Source Code

↓

Sonar Scanner

↓

SonarQube

↓

Quality Gate

↓

Jenkins Pipeline

<br>

## Quality Gate

<br>    

Quality Gate blocks deployment if:

- Bugs
- Vulnerabilities
- Code Smells
- Coverage
